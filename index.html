03 // COMPUTER VISION

精准激光除草机

这个项目将计算机视觉与精密硬件控制相结合，实现农业领域的自动化除草。我利用 OpenCV 进行实时图像采集与处理，并通过深度学习算法（如 YOLO 或自定义 CNN 模型）精准识别田间杂草的位置和种类。识别出的杂草坐标被传输给高功率激光器，实现毫米级的定点清除，有效减少农药使用，提升农业效率。

04 // GPS & HARDWARE

北斗无人船水质监测

致力于海洋环境保护。我参与开发了一款集成北斗定位系统的无人船，能够根据预设路径自主巡航，并实时采集水质数据（如 pH 值、溶解氧、浊度等）。核心控制采用 STM32 单片机，通过 LoRa 模块实现远距离数据传输，确保数据及时回传到监控中心，为水质监测和污染溯源提供精准数据支持。

05 // GENERATIVE ART

黎族文化 AIGC 传承
